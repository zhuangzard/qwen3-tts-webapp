**2. 感知系统的四层架构已清晰**

- **几何层**: Hough空间轴线检测（ROOM 2/Endoluminal）、圆柱体反演（手术器械位姿）；

#### 方法论归纳

**手术机器人AI的五条核心技术路线（2026年视角）**：

1. **仿真数据引擎路线** (ROOM / CathSim / SurgWorld)
   - CT/MRI重建解剖→物理仿真→光真实感渲染→合成数据训练
   - 频域噪声建模 > 简单高斯噪声（对单目深度估计关键）
   - Flow Matching世界模型（Cosmos-Predict2.5 LoRA）已达5条轨迹微调可用水准

2. **感知物理约束自监督路线** (PPR / Endoluminal)
   - 利用内窥镜物理特性（共轴光源、球面对称衰减）构造无监督信号
   - 合成数据Teacher + 临床数据Student自监督迁移 → 无需真实深度标注
   - Matrix Fisher不确定性量化 → 安全门

3. **逆映射控制路线** (LapSurgie / Contact-aware RRT)
   - 被动运动链的解析建模（余弦定理+约束优化）
   - 接触即导航：运动原语（壁面滑行/自由弹射/成角发射）代替高精度物理仿真
   - RCM约束内嵌于优化结构（而非运动规划层外挂约束）

4. **知识符号化+可满足性评估路线** (SurgGoal / Affordance-Based)
   - 手术规则形式化（Required/Allowed Steps + Dependencies + Prohibitive Orderings）
   - goal-satisfiability比序列相似度更适合安全关键评估
   - Conformal Prediction提供分布无关统计保证

5. **医疗语言模型工程化路线** (MedXIAOHE)
   - 实体树驱动的长尾覆盖（140万实体，Aho-Corasick O(N)扫描）
   - RFT增强课程RL解决稀疏奖励和熵坍塌
   - 多层混合奖励（规则+Rubrics+过程监督+二元安全门控）

#### 跨领域关联

- **SurgWorld IDM伪标签 + PhysRobot SV-pipeline**: SurgWorld解决了手术视频→运动学的无监督转化；若IDM预测的动作带有守恒律违约标记，则可用于PhysRobot的DPO偏好对（chosen=守恒律满足，rejected=守恒律违约）→ 手术数据飞轮自动生成PhysRobot训练数据。

- **Contact-aware RRT三原语 → PhysRobot组织穿刺路径**: 壁面滑行/自由弹射/成角发射对应软组织操作中的"沿组织表面滑行/跨越软组织自由段/成角插入深层结构"——物理接触约束是共同机制，运动原语抽象可直接迁移。

- **Matrix Fisher不确定性 → PhysRobot守恒律置信门**: 当PhysRobot GNN的守恒律违约率超过阈值时，从"自主执行"切换到"人工监督"模式；Matrix Fisher在SO(3)上的集中度可作为手术器械姿态控制的不确定性测度。

- **MedXIAOHE RFT增强课程RL → PhysRobot tissue_grasp训练**: 手术RL与机器人RL面临同样的稀疏奖励+熵坍塌问题；RFT Phase（从当前策略识别优质样本蒸馏）+ Foundation Phase（简单任务锚定）+ Specialization Phase（复杂物理约束场景）= 完整PhysRobot训练课程。

- **SurgiPose → 大规模手术示范数据**: 如果SurgiPose能从YouTube手术视频提取运动学轨迹，与PhysRobot结合：运动学轨迹+PhysRobot GNN推断每步接触力→生成(轨迹, 接触力, 守恒律满足性)三元组数据集，是手术物理先验数据飞轮的完整链。

- **Surgical-MambaLLM SIP扫描 → NIAF隐式动作场**: SIP从图像中心向外径向扫描，与NIAF在图像坐标系内构建C∞光滑动作场有几何类比——NIAF的动作场可以按照"器械目标位于中心，运动场从中心向外辐射"的先验设计初始化，提高SIREN学习效率。

#### 开放问题

**1. 手术仿真器的"充分保真度"定义**
- ROOM聚焦气道（纹理极少+镜面强烈），CathSim聚焦血管介入，SurgWorld聚焦da Vinci针操作——三个仿真器在各自场景达到实用阈值，但跨场景的"仿真器迁移"能力几乎零研究。一个仿真器能否通过LoRA/adapter快速迁移到新解剖结构？

**2. 从无标注视频到力反馈的完整桥梁**
- SurgiPose可从视频提取运动学，但无法提取接触力（力是内部状态）；IDM可推断动作，但仅限于视觉可见的EE运动；手术力觉数据仍是最大短缺——物理仿真（ROOM的PINN路线）+真实dVRK ATI力传感器标定是唯一现实路径，但数据量极少。

**3. 手术VLA的语言遵循评估空白**
- CAG研究显示通用VLA语言遵循率仅15-35%；SurgGoal显示最佳手术Video-LLM步骤识别率仅39.4%——但手术VLA（SurgWorld + GR00T N1.5）的语言遵循率从未被系统评估。手术语言("needle driver grasps the tissue on the left of the suture line")的精确遵循比通用语言更关键。

**4. 微型机器人的闭合控制回路**
- Miniature soft robot展示5种功能切换+6-DOF运动，但：工作空间仅1.6³ cm³（小于人体腔道），成像反馈（超声/MRI）尚未整合，自主导航尚未实现。从"有人控制"到"闭环自主"的距离仍非常大。

**5. 符合预测在手术AI中的统计保证范围**
- Affordance消歧用CP提供统计保证（90%分位置信度），但60%消歧率仍不足。CP的保证是"我不确定时会暂停"，但无法保证"我确定时一定正确"——对于手术场景，False Positive（错误执行）的后果比False Negative（不必要暂停）严重得多，CP的非对称成本问题未被解决。
